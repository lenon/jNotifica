O que é?

O jNotifica é um plugin de notificações pra jQuery (JS). Foi criado com o objetivo de exibir mensagens amigáveis ao usuário, para evitar o uso de alerts() por exemplo.

Funciona em quais navegadores?

É compatível com a maioria dos navegadores atuais (e o IE6):

  • Firefox
  • Opera
  • Google Chrome
  • Internet Explorer (6+)

Legal, é só "linkar" a biblioteca jQuery e já posso utilizá-lo?

Sim! O jNotifica não precisa de nenhum outro plugin, nem CSS extra. É só linkar o jQuery e o plugin que você já pode utilizar os métodos/funções disponíveis. :)

Quanto custa?

Nenhum tostão! Você pode utilizar à vontade, inclusive para fins comerciais. O plugin está sob uma licença GPL.

Encontrei um bug, como faço para reportá-lo?

Abra um issue no Github, descrevendo o bug e (se possível) como reproduzir o problema. Screenshots também ajudam um bocado. Ou, se preferir, pode mandar um email para contato(at)lenonmarcel.com.br.

Gostaria de colaborar com o projeto

Ótimo! Você pode criar um fork do projeto. Aí, quando criar algo legal e estiver pronto, é só solicitar um Pull Request no Github do jNotifica. :)
