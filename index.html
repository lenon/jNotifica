jNotifica

jQuery notifications plugin

Tested in following browsers:

  • Opera 10
  • Internet Explorer 6, 7 and 8
  • Firefox 3.5
  • Google Chrome 3
However, should work on other browsers. Or not...

Default usage:

$.jnotifica(MESSAGE, [TIMEOUT]) or $.jnotifica(MESSAGE, [OPTIONS, TIMEOUT])

Shortcut:

$.N(MESSAGE, [TIMEOUT]) or $.N(MESSAGE, [OPTIONS, TIMEOUT])

Examples:

Example 1 - Default options (without timeout)

Show notification

$.jnotifica('Hello, world!');
Example 2 - 5 seconds timeout

Show notification

$.jnotifica('Hello, world!',5000);
Example 3 - 5 seconds timeout, without close button

Show notification

$.jnotifica('Hello, world!',{ closeButton : false },5000);
Example 4 - Fade effect

Show notification

$.jnotifica('Hello, world!',{ effect : 'fadeIn', closeEffect : 'fadeOut' });
Example 5 - White background

Show notification

$.jnotifica('Hello, world!',{ effect : 'fadeIn', closeEffect : 'fadeOut', css: { background : '#fff', color : '#000', borderBottom : '2px solid #444' } });
