Lista de opções disponíveis

Opção Valor padrão Descrição
effect slide Efeito de animação da notificação, podendo ser 'slide', 'fade' ou 'none'.
speed 800 Tempo (em milissegundos) que o efeito de animação irá durar.
closeClick true A notificação deve ser fechada ao ser clicada? Valores aceitos são TRUE ou FALSE.
closeButton true O botão de fechar deve ser exibido? Valores aceitos são TRUE ou FALSE.
closeText 'fechar' Texto do botão de fechar.
css {
color : '#fff',
opacity : 0.8,
cursor : 'pointer',
background : '#000',
padding : 25,
fontSize : '15px',
fontFamily : 'Arial, sans-serif',
textAlign : 'left'
} 		Estilo da mensagem. Qualquer propriedade CSS é aceita.
closeCcss {
fontSize:'12px',
color:'#fff'
} 		Estilo do botão de fechar. Qualquer propriedade CSS é aceita.
openCallback function(){} Função de callback que será chamada ao abrir a notificação.
callback function(){} Função de callback que será chamada ao fechar a notificação.
