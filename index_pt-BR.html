Um plugin de notificações pra jQuery

English version of this page →

O que é?

O jNotifica é um plugin pra jQuery que provê um sistema bonito e simples de notificações. É útil para substituir os alerts do JavaScript.

Funciona em quais navegadores?

O jNotifica é compatível com a maioria dos navegadores modernos:

  • Firefox
  • Opera
  • Google Chrome
  • Internet Explorer 7
  • Internet Explorer 8

E também o Microsoft Internet Explorer 6, mas...

...é um suporte ruim, pelo fato de o IE6 não suportar a propriedade CSS position: fixed. Precisei utilizar uma "gambiarra" para oferecer o suporte ao IE6, e essa solução não provê uma boa "experiência do usuário". Mas funciona.

Alguns exemplos:

- Mensagem no topo 

$.jnotifica('Hello from Brazil!',{
  margin : 10,
  width  : 400,
  effect : 'fade',
  padding: 25,
  msgCss : {
    textAlign : 'center',
    fontSize  : '138.5%',
    fontWeight: 'bold'
  }
});

- Mensagem no canto inferior direito 

$.jnotifica('Hey! You here? ;-D',{
  margin    : 10,
  position  : 'bottom',
  align     : 'right',
  background: 'purple',
  width     : 400
});

- Mensagem grande, na parte inferior 

var timer, color;
$.jnotifica('BUY ME! BUY ME!',{
  position  : 'bottom',
  background: 'navy',
  onShow    : function(Main){
    timer = setInterval(function(){
      Main.find('.jnotifica_bg').css('backgroundColor',color?'red':'navy');
      color = color ? false : true;
    },200);
  },
  onClose   : function(Main){
    clearInterval(timer);
  }
});

- Mensagem com HTML HI! 

$('span.example_4').jnotifica({
  position  : 'bottom',
  background: 'green',
  clickClose: false,
  timeout   : 0,
  cursor    : 'default'
});

- Cantos arredondados 

$.jnotifica('Beautiful! *-*',{
  position  : 'bottom',
  width     : 300,
  msgCss    : {
    textAlign: 'center'
  },
  background: 'blue',
  opacity   : 0.5,
  margin    : 30,
  classes   : 'rounded_corners'
});

Uso básico:

Exibir notificação:

$.jnotifica(MENSAGEM[, OPÇÕES]); ...ou: $.jN(MENSAGEM[, OPÇÕES]);

Fechar a notificação:

$.jnotifica.close(); ...ou: $.jN.close();

Notificação a partir de um elemento:

$(ELEMENT).jnotifica([OPTIONS]); ...ou: $(ELEMENT).jN([OPTIONS]);

Lista de oções:

Opção Valor padrão
margin 0 Em pixels
position top Posição da notificação, é aceito 'top' ou 'bottom'
width all Largura em pixels, ou 'all' para 100% de largura
align center 'center', 'left' ou 'right' - usado somente quando width != 'all'
padding 25 Em pixels
background #000
zIndex 100
color #fff Cor do texto
opacity 0.8 Opacidade do background
cursor pointer Cursor do mouse
msgCss {
fontSize: '15px',
fontFamily: 'Arial, sans-serif',
textAlign: 'left'
}		 CSS para a mensagem
effect slide Efeito para abrir/fechar - aceito: 'slider', 'fade' ou 'none'
speed 500 Velocidade do efeito
timeout 5000
clickClose true Clicar para fechar?
close {
text: '[CLOSE]',
css: {
color: '#fff',
fontSize: '10px',
position: 'absolute',
top: 5,
right: 10,
cursor: 'pointer'
}
}		 Propriedades do botão "Fechar", ou FALSE para não exibir o botão
onShow function(main){} Função Callback chamada ao abrir a notificação
onClose function(){} Função Callback chamada ao fechar a notificação
classes '' Classes extra para a DIV principal
