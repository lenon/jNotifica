Exemplos

  • Normal, sem nenhuma opção $.jnotifica();
  • Com mensagem personalizada $.jnotifica('Olha só, funciona! \\o/');
  • Timeout de 4 segundos $.jnotifica('Fechando em 4, 3, 2... :P',{},4000);
  • Sem timeout $.jnotifica('Enquanto você não clicar, a mensagem vai continuar aqui',{},0);
  • Com efeito fade ao invés de slide $.jnotifica('Opa',{effect: 'fade' });
  • Sem efeito $.jnotifica('Prefiro com efeitos =p',{effect: 'none' });
  • Sem botão de fechar $.jnotifica('Clique para fechar ;-)',{closeButton: false });
  • Fundo vermelho $.jnotifica('DANGER! DANGER! DANGER! DANGER!',{css : { background: 'red' } });
  • Fundo verde, fonte 14px e texto no centro $.jnotifica('Salvem as baleias.',{css : { background: 'green', fontSize : '14px', textAlign : 'center' } });
  • Sem opacidade e texto no centro $.jnotifica('Yeahh.',{css : { opacity: 1, textAlign : 'center' } });
  • Não pode ser fechado antes do timeout (10 segundos) $.jnotifica('Aguarde o timeout :)',{closeButton : false, closeClick: false}, 10000);
  • Exemplo da função de callback $.jnotifica('Buuu!',{callback : function(){ alert('Fechou :P')}}, 5000);
